# [ggoral.github.io](https://ggoral.github.io)

Welcome this is Gonzalo Goral's personal website

## Instruction

 ```sh
 gem install jekyll bundler
 git clone git@github.com:ggoral/ggoral.github.io.git
 cd ggoral.github.io
 bundle install
 bundle exec jekyll build
 ```
